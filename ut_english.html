6A Past Perfect Tense

{}=Past Perfact,[]=Simple Past

1.Use Before or After with {Past Perfect} and [Simple Past Tenses] to talk two [PAST] actions.

(Use Past Perfect refer to First action,Simple Past Tense to refer Second one)

Ep1 : The man {had gone} before she [opened] the door

Ep2 : Miss Lee [emaild] us after she {had received} the parcel

2.Negative statements

(Add NOT+Past Perfect or short forms)

Ep1 : The passenger [missed] the plane because she {had not brought} her passport.

Ep2 : When my dad [was] 11,he still {had not been} to the beach.

3.Yes/No Questions

ASK:

(Had + (I/you/we/they/he/she/it) + Pass Perfact(Verb))

Ep1 : Had they worken up?

Ep2 : Had the police caught the thief?

ANSWER:

(Yes/No + (I/you/we/they/he/she/it) + had/had not(hadn't))

Ep1 : Yes,they had

Ep2 : No,he had not

4.Wh Questions

(what/where/who/when/why/which/how + had + I/he/she/it/you/we/they + Past Perfact(Verb))

Ep1 : What had he told you?

Ep2 : who had the woman talked to?

5.Past Perfect with Wish/If only

WISH:

(I + wish + (I/we/they/he/she/it) + had/had not + Past Perfact )

Ep1 : I wish I had passed the exam.

Ep2 : I wish we had found the map earlier

IF ONLY:

(If only + (I/we/they/he/she/it) + + had/had not + Past Perfact)

Ep1 : If only he had told us the truth.

Ep2 : If only I had not bought the house.

CHANGE:

Ep1 : (If only my broyher had passed the exam.)-->(I wish my brother had passed the exam.)

6C Using to-infinitives and gerunds

1.The Verbs We must use to-infinitives

[agree] [choose] [learn] [proceed] [tend] [aim] [decide] [need] [promise] [volunteer] [ask] [fail] [plan] [refuse] [want] [attempt] [hope] [prepare] [seem] [wish]

2.The Verbs We must use gerunds

[admit] [deny] [finish] [mind] [recommend] [appreciate] [dislike] [imagine] [miss] [resist] [avoid] [enjoy] [keep] [practise] [risk] [consider] [feel] [mention] [prevent] [suggest]

3.The Verbs We can use both

[begin] [hate] [like] [prefer] [continue] [intend] [love] [start]

4.The Verbs We can use both But each means different

(remember)and(forget):

+ to-infinitive (to talk about something we still need or needed to do)

+ gerund (to talk about something we did in the past)

regret:

+ to-infinitive (to announce bad news)

+ gerund (to talk about something that we are sorry we did)

go on:

+ to-infinitive (to talk about stopping an activity and switching to another one)

+ gerund (to talk about continuing an activity without stopping)

stop:

+ to-infinitive (to talk about stopping an activity in order to do something else)

+ gerund (to talk about stopping an activity)

try:

+ to-infinitive (to talk about making an effort to do something)

+ gerund (to talk about doing an experiment to see what will happen)

8D Defining and non-defining relative clauses

1.Defining

Defining relative means that if we remove a defining relative clause from a sentence,the main ider will change

Relative pronoun

Who/that == people
which/that == things
whose == possession

Ep1 : The boss like employees (who are organized)

(means The boss likes organized employees)

2.non-Defining

non-defining relative means that even we remove a defining relative clause from a sentence,the main ider won't change

Relative pronoun

Who == people
which == things
whose == possession

Ep1 : The boss likes Jack,(who is an organized employee)

(means The boss likes Jack)
